Featured Projects

Tanglish Programming IDE

A custom IDE that allows writing Python code using Tamil-English mixed syntax (Tanglish), making programming more accessible to Tamil-speaking beginners.

Key Features & Technologies:

  • Supports mixed Tamil-English syntax for Python code with real-time translation
  • Built with Python and Tkinter for the GUI components
  • Custom lexer and parser for Tanglish syntax interpretation
  • Interactive console with error detection and highlighting
  • Beginner-friendly interface with code suggestions
  • Export functionality to standard Python files

The IDE translates Tanglish code to standard Python and executes it, with clear output display. This helps bridge the gap for Tamil-speaking learners transitioning to English-based programming. The project received positive feedback from local coding bootcamps and educators.
